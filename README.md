# 🚀 Blog Application

A full-stack blog application built using **Node.js, Express, MongoDB, and EJS**.
This project allows users to create, read, update, and delete blog posts with additional features like likes, comments, and search functionality.

---

## 📌 Features

* ✍️ Create, edit, and delete blog posts
* 🔍 Search blogs by title
* ❤️ Like system for each blog
* 💬 Comment system
* 📱 Responsive user interface
* 🧠 Clean and structured backend (MVC style)

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, EJS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Tools:** Git, GitHub

## ⚙️ Installation & Setup

1. Clone the repository
   git clone https://github.com/Yash-Patil123/blog-app.git

2. Navigate to the project folder
   cd blog-app

3. Install dependencies
   npm install

4. Start the server
   node app.js

5. Open in browser
   http://localhost:8080
